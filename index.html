雲端模式（Supabase，自動上傳）

填入 Project URL 與 Anon Key 後按「儲存並連線」。之後新增/修改/刪除紀錄與決策，會自動同步到雲端。

設定步驟

1. 登入 Supabase - 前往 supabase.com

2. 建立新專案 - 或使用現有專案

3. 複製 Project URL - 在 Settings → API 中找到

4. 複製 Anon Key - 在 Settings → API 中找到

5. 貼到上方欄位 - 然後點「儲存並連線」

6. 建立資料表 - 首次連線時，系統會提示您需要執行 SQL，複製 SQL 到 Supabase SQL Editor 執行即可
未連線 - 資料存儲在本機
